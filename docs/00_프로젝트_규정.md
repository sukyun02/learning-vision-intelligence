---

## 1. 데이터셋

- **CIFAR-100만 사용** (train 50,000장 / test 10,000장)
- 외부 데이터, 외부 pretrained weight 일체 사용 불가
- Test time adaptation 기법 사용 불가

---

## 2. 학습 방식

- **From scratch** 학습만 허용 (사전학습 weight 사용 금지)
- 오픈소스 코드를 단순 복붙하면 감점 → 모든 구성요소를 이해하고 있어야 함

---

## 3. 학습 환경 및 시간

- **모델 1개당 single GPU (렌탈)** 사용 — **RTX A5000 (VRAM 24GB)**
- **모델 1개당 학습 시간 1일(24시간) 이내** (approximately)
- TA 및 피어리뷰 기간 중 검증됨

---

## 4. 평가 지표 (2개)

### 4-1. Top-1 Accuracy
- Fine-class 100개 기준 Top-1 정확도

### 4-2. Super-Class Accuracy (SC Density)
- Top-5 예측 fine class 중 정답 superclass에 속하는 클래스 비율
- 예시: GT=Maple(Trees), Top-5=[Maple, Oak, Pine, Apple, Rose] → 3/5 = 0.6
- **측정 함수를 팀이 직접 구현해서 제출** (첫 번째 팀 과제)
- 20 superclass × 5 fine class 구조 활용

---

## 5. 재현성

- **3개 이상의 random seed로 실험** → mean ± std 보고
- 피어리뷰 팀이 새로운 랜덤 시드로 재현 테스트 수행
- TA도 별도로 재현 검증

---

## 6. 피어리뷰

- 모든 팀은 다른 팀 2곳의 코드를 재현하여 검증
- **최종 점수 = 피어리뷰 팀(2팀) + TA가 재현한 결과**로 결정
- 피어리뷰 기간 중 학습 환경(single GPU) 및 시간(24h/모델) 조건도 함께 검증됨
